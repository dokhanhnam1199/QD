```python
import numpy as np

def priority_v2(item: float, bins_remain_cap: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    The priority is calculated based on the remaining capacity of each bin.
    We prefer bins that have just enough capacity to hold the item.
    """
    # Calculate the priority score for each bin
    # The score is high if the bin has just enough capacity to hold the item
    priority = np.where(bins_remain_cap >= item, 
                        1 - np.abs(bins_remain_cap - item) / item, 
                        0)
    return priority
```
